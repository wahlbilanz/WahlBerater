# WahlBerater

![WahlBerater Logo](src/assets/static/wahlberater.png)

Der WahlBerater ist eine technische Wahlentscheidungshilfe.
Wählende sollen leicht ihre Position mit der Position von Kandidierenden vergleichen können.
